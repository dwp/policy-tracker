Information gather

In order to start prototyping we needed to start gathering real content which would be relevant to the pages we were going to design.

Activity

Again we created columns on our brown paper and used green post-its to label them. We wanted to find out about the different things that triggered work, the different areas (directorates) the work were done in, the different factors which effected how work is prioritised and the actually different types of work that was done.

Results

We now had a lot of the details we needed to start designing pages.
